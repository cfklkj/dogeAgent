# 📝 更新日志

## [3.0.1] - 2026-03-10

### ✨ 新增功能
- **拖动功能**: 现在可以通过拖动狗子头像移动窗口位置
- **双击交互**: 双击狗子打开聊天窗口
- **拖动提示**: 启动时显示操作提示（3 秒后自动消失）

### 🐛 Bug 修复
- 修复了窗口无法拖动的问题
- 优化了拖动体验，添加视觉反馈
- 修复了动画与拖动冲突问题

### 🎨 优化改进
- 拖动时暂停待机动画，提升视觉体验
- 添加拖动状态指示（爪子图标）
- 优化了双击和拖动的区分逻辑

---

## [3.0.0] - 2026-03-10

### ✨ 核心功能
- 完整的 Electron + Python 架构
- NVIDIA z-ai/glm5 和 Google Gemini 双模型支持
- 情感引擎系统（亲密度、性格、情感状态）
- 工具系统（时间、日期、计算器、天气）
- 语音识别和合成框架
- 插件系统基础框架

### 🎨 UI/UX
- 透明窗口桌面宠物
- 毛玻璃效果 UI
- 深色主题
- 系统托盘集成
- 右键菜单

### 📦 技术栈
- Electron 28.3.3
- LangChain
- Python 3.10+
- SQLite 数据存储
