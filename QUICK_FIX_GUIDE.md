## 问题：聊天窗口显示"未连接"

### 症状

- 或者一直显示 "连接中..."

---

## 解决方案

### 方案 1: 重启应用（推荐）

1. 完全退出应用
2. 重新运行：
```bash

```

### 方案 2: 检查 Python 服务

```bash
# 测试 Python 桥接
python test_bridge.py
```

如果测试失败，检查：
- Python 依赖是否安装
- `.env` 文件是否配置
- NVIDIA API Key 是否有效

### 方案 3: 查看详细日志

启动时添加日志：
```bash
npm start --enable-logging
```

查看输出中的错误信息。
