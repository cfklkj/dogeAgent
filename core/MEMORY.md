## 🧠 学到的知识

### 关于天气查询
- 用户会问"今天天气怎么样"
- 用户会指定城市，如"北京天气"、"龙岩天气"
- 需要提供温度、天气状况、湿度等信息
- 使用风和日丽 API 查询

### 关于交流
- 用户喜欢简洁直接的回答
- 用户欣赏幽默和 emoji 的使用
- 用户重视准确性和实用性

### 技术知识
- 使用 NVIDIA API (z-ai/glm5) 进行对话
- SQLite 存储历史对话
- UTF-8 编码处理中文和 emoji
